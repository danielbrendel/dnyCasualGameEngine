Casual Game Engine is a top down 2D singleplayer shooter with moddable content

the game consists of the core engine and the mod package which is loaded via the menu
a package is a complete mod as well as a campaign that can consist of:
- player definition (model, attributes)
- maps which define the landscape, obstacles and entities
- entity scripts for custom entities in the world
- graphics and sounds

A package can be loaded via the main menu and then played. the default goal is
to run through the map from start to finish and fight against the opponents that
want to prevent you from finishing the goal. 

The game shall be available on Steam featuring the Workshop in order to let users publish their packages
which can then be subscribed to and automatically downloaded to the game.
